## Task Scheduler

Реализация простого многопоточного планировщика задач на C++

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/KrivosheevNikita/cpp-task-scheduler.git
cd cpp-task-scheduler
```

### 2. Сборка и запуск
```bash
mkdir build
cd build

cmake ..
cmake --build .

./task_scheduler
```
